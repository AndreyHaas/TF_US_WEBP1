✨ Best Practices der Woche

📦 Builder-Pattern

Bei 5+ Parametern nutze Builder statt verschachtelter Konstruktoren.

⚡ Optional nicht für Felder

Optional nie als Feld-Typ verwenden – es bricht die Serialisierung.

🔄 Immutable Collections

List.copyOf() statt Collections.unmodifiableList() für bessere Performance.

⚙️ Pattern Matching für instanceof

Seit Java 16 spart Pattern Matching Boilerplate-Code und erhöht die Lesbarkeit.
