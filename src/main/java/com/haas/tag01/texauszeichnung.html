Texte auszeichnen

Zeilenumbrüche werden im HTML-Markup ignoriert, aber bestimmte tags bringen einen Zeilenumbruch mit. Dazu gehören die tags: h1-h6 und p (Absatz). Mehr als ein Leerzeichen wird vom browser ignoriert. Das p-tag erzeugt einen Absatz und einen Abstand nach oben und unten.

Zeilenumbruch (br)
Das br-Element (break, englisch für Unterbrechung) repräsentiert einen logischen Bruch innerhalb eines Textes. Visuell wird standardmäßig ein Zeilenumbruch gesetzt.

Horizontale Linie

Das tag hr erzeugt eine horizontale Linie, die über die gesamte Breite des Browserfensters geht. br ist ein stand-alone-tag, es gibt kein end-Tag.

Betont (em) Mit dem Element em wird ein Teil eines Fließtextes als betont (engl. emphasized) ausgezeichnet.

Stark betont (strong) Mit dem Element strong wird ein Teil eines Fließtextes als stark betont (engl. strongly emphasized) ausgezeichnet.
