Jedes HTML-Element ist eine Box

Jedes HTML-Element besitzt Ausmaße: Breite (width), Höhe (height), Innenabstand (padding), Rahmen (border) und einen Abstand nach Außen (margin). Diese sind schon im Browserstylesheet vordefiniert, können aber verändert werden.

Viele Elemente beanspruchen eine komplette Zeile im Browser für sich und lassen kein weiteren Elemente neben sich zu. Dies ist geregelt mit der Eigenschaft display (Wie erscheine ich). Der Wert block (Ganze Browserbreite) ist standardmäßig für viele HTML-Elemente im Browser-Style- Sheet gesetzt. Mit dem Wert inline kann man Elemente nebeneinander bekommen.
