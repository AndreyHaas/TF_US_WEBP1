Zwei div-Container untereinander

Jedes HTML-Element – gleich ob body, div, p, table, a oder span – wird im Box-Modell als rechteckige Box betrachtet. Das Box-Modell mit padding, border und margin ist die Grundlage für das grafische Layout und die Berechnung der Breite und Höhe von Elementen mit CSS. Die einfache Angabe der Breite allein reicht allerdings nicht, um den Raum für ein Layout-Element zu berechnen.
Jedes Block-Element (div, p, table, …) dehnt sich über die gesamte Breite der umfassenden Box aus, wenn width für das Element nicht angegeben ist. Inline-Elemente wie a, span und strong nehmen innerhalb der umfassenden Box genau den Platz für ihren Inhalt ein. Ein div-container ist eine Box für Inhalte und hat keine eigenen Eigenschaften. Div-container werden fürs Layout verwendet.
