Elemente auf der Kreuzachse ausrichten

Align-items richtet die Elementen vertikal auf der Kreuzachse aus (hir ist das die Kreuzachse, da die Hauptachse horizontal ist). align-items: flex-start; richtet die Elemente am Anfang der Kreuzachse aus, align-items: center; richtet die Elemente in der Mitte der Kreuzachse aus und align-items hat einen Standartwert: stretch, damit werden die Elemente auf die volle Höhe der Kreuzachse gestreckt. Angabe werden die Kindelemente so hoch wie das Elternelement, damit die Kreuzachse komplett ausgefüllt wird.
